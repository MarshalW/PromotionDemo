# PromotionDemo

使用React实现的翻页动画原型

## 安装

需要安装`nodejs`

需要安装`gulp`：

```
npm install --global gulp
```

然后，执行：

```
git clone git@github.com:MarshalW/PromotionDemo.git
```

进入`PromotionDemo`目录， 切换到react分支，执行：

```
git branch react
```

然后安装所需的包，执行：

```
npm install
```

## 运行

执行gulp任务，生成所需的文件：

```
gulp
```

然后用浏览器打开`index.html`文件，看是否正常。
